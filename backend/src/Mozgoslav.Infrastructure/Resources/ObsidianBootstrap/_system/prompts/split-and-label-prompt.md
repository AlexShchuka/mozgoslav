# ЗАДАЧА

Дан сырой текст (см. секцию ВХОДНОЙ ТЕКСТ ниже). Сделай две вещи разом:

1. Применить словарь корректировок (см. секцию СЛОВАРЬ) к тексту — заменить искажённые слова на правильные ДО анализа.
2. Разбить текст на тематические блоки и пометить каждый лейблом.

# ЛЕЙБЛЫ

- **ИДЕЯ** — конкретное предложение, мысль, потенциальное решение, гипотеза. Что-то новое, что можно потом развивать.
- **ИНСАЙТ** — наблюдение, вывод о том как что-то устроено или работает. Понимание, не план.
- **ЧЕЛОВЕК** — упоминание конкретного человека и контекст вокруг него (что сказал, что сделал, какое отношение к теме).
- **ВОПРОС** — открытый вопрос без ответа, что-то непонятое, что хочется выяснить.
- **ЗАДАЧА** — конкретное действие, которое нужно сделать. Императивная форма.
- **FLAGGED** — слова или фразы, которые непонятны по смыслу, выпадают из контекста, выглядят как ошибка распознавания и НЕТ в словаре корректировок.

# ФОРМАТ ВЫВОДА

Каждый блок начинается с H2 заголовка вида:
```
## ЛЕЙБЛ: краткое название (до 8 слов)
текст блока в 1-3 предложениях
```

Между блоками — одна пустая строка. Других заголовков (#, ###) не используй.

Для FLAGGED формат проще:
```
## FLAGGED: <слово или фраза>
короткий комментарий: где встретилось, что вероятно имелось в виду
```

# ПРАВИЛА

1. **Сохраняй контекст.** Если идея стоит на чём-то сказанном до неё — включи это в текст блока, чтобы он читался самостоятельно.
2. **Не дублируй.** Одна мысль — один блок. Если тема всплывает дважды — слей в один блок.
3. **Не выдумывай.** Если в тексте нет блоков какого-то типа — просто не выдавай этого H2. Не нужно «придумать вопрос ради формата».
4. **Не сокращай агрессивно.** Лучше передать суть в 3 предложениях, чем потерять её в 1.
5. **Только русский язык.** Без английских слов в выводе. Если в исходнике английский термин и его уже нет в словаре — оставь как есть, но добавь рядом краткий русский эквивалент в скобках.
6. **Слова-паразиты убирай.** «Ну», «типа», «короче», «вот», «значит» — не переносить в финальные блоки.
7. **Лейблы строго заглавными.** ИДЕЯ, не Идея и не идея.
